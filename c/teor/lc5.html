Урок №5. Что такое CSS?

Добро пожаловать в CSS!

CSS означает Каскадные Таблицы Стилей (Cascading Style Sheets).

Каскадные относится к способу применения CSS одного стиля к вершине другого.
Таблицы Стилей управляют отображениемweb-документов.

CSS и HTML работают рука об руку:
HTML сортирует структуру страницы.
CSS определяет, как будут отображены элементы HTML.

Чтобы понять CSS, пользователю необходимо иметь базовые знания HTML.
Пользователю предоставляется доступ к изучению HTML в данном web-приложении.

Для чего используется CSS?

CSS позволяет вам применять специфические стили к специфическим HTML элементам.

Основным преимуществом CSS является то, что он позволяет разделять стиль от содержимого.

Используя только HTML, все стили и форматирование будут в одном месте, что приведет к намного более сложному управлению страницей при ее росте.

Все форматирование можно убирать из HTML документа и размещать в отдельном CSS файле.
