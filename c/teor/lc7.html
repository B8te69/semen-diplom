Урок №7. CSS правила и Селекторы

Синтаксис CSS

CSS состоит из правила стиля, которые браузер интерпритирует и затем применяет к соответствующим элементам в вашем документе.
Каждое правило имеет три части: селектор, свойство и значение.
Селектор указывает на элемент HTML, стиль которого пользователь хочет изменить.

Например, цвет селектора может быть определен следующим образом:





Блок объявлений содержит одно или несколько объявлений, разделенных точкой с запятой.
Каждое объявление включает в себя имя свойства и значение, разделенные точкой с запятой.
