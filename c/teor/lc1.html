Урок №1. Знакомьтесь: HTML!

Аббревиатура HTML означает "Hyper Text Markup Language", или "язык гипертекстовой разметки".

В отличие от языков программирования (в том числе скриптовых), в которых с помощью сценариев программируются функции, в языке разметки с помощью тегов программируется контент (содержимое).



Данный курс состоит из интерактивных тестов и упражнений, которые помогут вам быстро и эффективно освоить материал.



Структура web-страницы

Любому уважающему себя web-дизайнеру необходимо научиться программировать на языке HTML. Более того, именно со знания HTML должно начинаться любое знакомство с премудростями web-дизайна.

Web-дизайн сегодня:
HTML - структура
CSS - оформление
JavaScript - поведение
PHP и похожие языки - серверная часть
CMS - управление контентом

Язык HTML легко выучить. Поэтому приступайте к его обучению прямо сейчас!
